**Build Instructions for Faucet Script**

1. Assuming you have nodeJS and npm installed, run 'npm install' inside the kuwa_faucet_script dir to install all the required modules.

2. Update config.json file with the correct values and addresses(present in TCUP google drive folder)

3. run 'node kuwa_faucet_script.js' to trigger the payment process

Additional info:
Once the transfers start happening, you can search for the wallet address on etherscan and see the txn. Also, you can load the wallet into metamask and add the KuwaCoin contract address to track KuwaCoin balance of the account
